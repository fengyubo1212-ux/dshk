第3章 生命起源和生物进化

一、生命的起源

  • 化学起源说（主流学说）：原始地球上简单的无机物在特定条件（高温、紫外线、雷电）下，经过漫长的化学演化过程，逐渐生成有机物，最终形成原始生命。
  • 原始地球条件：原始大气中含有甲烷、氨、氢气、水蒸气等，没有氧气。温度很高，有强烈的紫外线和频繁的雷电。
  • 米勒实验：美国学者米勒模拟原始地球条件和大气成分，将甲烷、氨、氢气、水蒸气等混合，通过火花放电（模拟雷电），合成了氨基酸等有机小分子。证明了：原始地球上可以由无机物合成有机小分子。
  • 原始生命诞生的标志：原始生命能进行生长、生殖、遗传等特征。
  • 原始海洋是生命的摇篮。

二、生物进化的历程

  • 进化总趋势：由简单到复杂、由低等到高等、由水生到陆生。
  • 植物进化历程：原始藻类 → 苔藓植物 → 蕨类植物 → 裸子植物 → 被子植物。
  • 动物进化历程：原始单细胞动物 → 腔肠动物 → 扁形动物 → 线形动物 → 环节动物 → 软体动物 → 节肢动物 → 鱼类 → 两栖类 → 爬行类 → 鸟类和哺乳类。
  • 化石证据：越古老的地层中，化石生物越简单、越低等；越晚近的地层中，化石生物越复杂、越高等。化石是研究生物进化最直接、最重要的证据。
  • 重要过渡类型化石：始祖鸟化石（爬行动物向鸟类进化的证据）。

三、自然选择学说

达尔文的自然选择学说是生物进化理论的核心，包括四个要点：

  • 过度繁殖：自然界中的生物普遍具有很强的繁殖能力，产生大量后代。
  • 生存斗争（生存竞争）：由于食物、空间等资源有限，大量后代必须为生存而进行斗争。
  • 遗传变异：后代中存在可遗传的变异，有些变异对生存有利，有些不利。
  • 适者生存：在生存斗争中，具有有利变异的个体能更好地适应环境，生存下来并繁殖后代；具有不利变异的个体则被淘汰。经过长期的自然选择，有利变异不断积累，形成适应环境的性状。

实例——长颈鹿的进化：古代长颈鹿存在颈长差异（变异）→ 环境变化，低处食物减少 → 颈长的个体能吃到高处树叶，容易生存 → 颈长的个体繁殖后代 → 经过漫长自然选择，长颈性状积累，形成现在的长颈鹿。

自然选择是生物进化的根本原因。

四、人工选择

人工选择：人类根据自身需求，通过选择和培育，使生物向人类所需要的方向发展。如：各种品种的鸡（蛋鸡、肉鸡）都是通过人工选择从原鸡培育而来的。

对比项目 自然选择 人工选择
选择因素 自然环境 人类需求
选择速度 非常缓慢（需漫长时间） 相对较快
选择方向 有利于生物适应环境 满足人类需求（可能不利于生物自身）
结果 形成适应环境的特征，促进生物进化 培育出满足人类需要的新品种
实例 长颈鹿的长颈、枯叶蝶的拟态 蛋鸡、肉鸡、各品种犬、金鱼
