第1章 生物的生殖和发育

一、有性生殖与无性生殖

有性生殖：由两性生殖细胞（精子和卵细胞）结合形成受精卵，再由受精卵发育成新个体的生殖方式。后代具有父母双方的遗传特性，变异性大，更适应环境变化。

无性生殖：不经过两性生殖细胞的结合，由母体直接产生新个体的生殖方式。后代与母体的遗传物质完全相同，能保持母体的优良性状，繁殖速度快。

二、植物的无性生殖

  • 扦插——剪取植物的茎、叶、根等插入土壤或水中，使其生根发芽成为新植株。常见：月季、柳树、葡萄。
  • 嫁接——把一种植物的芽或枝（接穗）接到另一种植物的茎或根（砧木）上，使它们愈合成一株新植株。嫁接成功的关键：接穗和砧木的形成层紧密结合。常见：果树改良。
  • 压条——将枝条压入土中使其生根，然后与母株分离成为新植株。常见：桂花、石榴。
  • 组织培养——利用植物组织或细胞，在无菌条件下培养成完整植株。优点：繁殖速度快、不受季节限制、可大量繁殖优良品种、脱毒苗培养。

无性生殖的优势：保持母体优良性状、繁殖速度快、后代性状一致。

对比项目 扦插 嫁接
方法 剪取茎、叶、根插入基质 将接穗接到砧木上
关键 保持湿润、适当遮阴 形成层紧密结合
常见应用 月季、柳树、葡萄 苹果、梨、桃等果树改良

三、昆虫的生殖和发育

昆虫的生殖：有性生殖、体内受精、卵生。

对比项目 完全变态发育 不完全变态发育
发育过程 卵 → 幼虫 → 蛹 → 成虫（四个阶段） 卵 → 若虫 → 成虫（三个阶段）
幼虫与成虫差异 形态结构和生活习性差异很大 若虫与成虫形态相似，仅生殖器官未成熟
是否有蛹期 有蛹期 无蛹期
代表昆虫 蚕、蝴蝶、苍蝇、蚊子、蜜蜂 蝗虫、蟋蟀、蝉、蜻蜓、螳螂

四、两栖动物的生殖和发育

  • 生殖特点：有性生殖，体外受精，水中进行。青蛙在繁殖季节有鸣叫和抱对行为（抱对能提高受精率）。
  • 发育特点：变态发育——幼体（蝌蚪）与成体（青蛙）在形态结构和生活习性上有显著差异。
  • 发育过程：受精卵 → 蝌蚪（用鳃呼吸，有尾，生活在水中）→ 幼蛙（长出四肢，尾消失，鳃退化，肺形成）→ 成蛙（用肺和皮肤呼吸，水陆两栖）。
  • 环境要求：两栖动物的生殖发育离不开水，因此对环境变化敏感，是环境监测的指示生物。

五、鸟的生殖和发育

生殖特点：有性生殖，体内受精，卵生。鸟类的生殖和发育过程一般包括：求偶 → 交配 → 筑巢 → 产卵 → 孵卵 → 育雏。

鸟卵结构 功能
卵壳 保护内部结构，上有气孔进行气体交换
卵壳膜 保护作用（内有外层和内层两层）
卵白（蛋清） 为胚胎发育提供水分和营养物质，也有保护作用
卵黄（蛋黄） 为胚胎发育提供主要营养物质
胚盘 含有细胞核，受精后发育成雏鸟（是胚胎发育的部位）
系带 固定卵黄，使胚盘始终朝上
气室 为胚胎发育提供氧气（位于卵的钝端）
