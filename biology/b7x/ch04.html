第4章 人体内物质的运输

一、血液的组成

血液由血浆和血细胞（红细胞、白细胞、血小板）组成。

成分 形态结构 主要功能
血浆 淡黄色液体，约占血液55% 运输血细胞，运输营养物质和废物
红细胞 无细胞核，含血红蛋白，呈红色 运输氧气（血红蛋白+O₂）
白细胞 有细胞核，比红细胞大 吞噬侵入人体的病菌，防御保护
血小板 无细胞核，最小，不规则 止血和凝血

贫血：血液中红细胞数量过少或血红蛋白含量过低。应多吃含铁和蛋白质丰富的食物。

发炎：人体某部位有炎症时，白细胞数量会增多。

二、三种血管

比较项目 动脉 静脉 毛细血管
管壁 厚、弹性大 较薄、弹性小 极薄（一层上皮细胞）
管腔 较小 较大（四肢静脉有静脉瓣） 极小（只允许红细胞单行通过）
血流速度 快 较慢 最慢
血流方向 心脏→全身 全身→心脏 连接微小动脉和静脉
瓣膜 无 有静脉瓣 无

静脉瓣的作用：防止血液倒流（四肢静脉内）。

三、心脏的结构

心脏主要由肌肉组织（心肌）构成，是血液循环的动力器官。

与心房相连的血管 与心室相连的血管
左侧心脏 左心房 ← 肺静脉（动脉血） 左心室（壁最厚） → 主动脉（动脉血）
右侧心脏 右心房 ← 上、下腔静脉（静脉血） 右心室 → 肺动脉（静脉血）

四个腔：左心房、左心室、右心房、右心室（上房下室，左右两侧不相通）。

瓣膜：

  • 房室瓣：心房与心室之间，防止血液从心室倒流回心房。
  • 动脉瓣：心室与动脉之间，防止血液从动脉倒流回心室。
  • 瓣膜只能单向开放，保证血液沿一定方向流动：心房 → 心室 → 动脉。

壁最厚的腔：左心室（因为要将血液泵到全身各处，需要最强收缩力）。

四、血液循环

体循环（大循环）：

左心室 → 主动脉 → 各级动脉 → 全身毛细血管 → 各级静脉 → 上、下腔静脉 → 右心房

在全身毛细血管处：血液与组织细胞进行物质交换——氧气和营养物质进入组织细胞，二氧化碳等废物进入血液。动脉血变为静脉血。

肺循环（小循环）：

右心室 → 肺动脉 → 肺部毛细血管 → 肺静脉 → 左心房

在肺部毛细血管处：血液与肺泡进行气体交换——二氧化碳进入肺泡，氧气进入血液。静脉血变为动脉血。

血液循环路线速记：

左心室 → 主动脉 → 全身（体循环）→ 上下腔静脉 → 右心房 → 右心室 → 肺动脉 → 肺（肺循环）→ 肺静脉 → 左心房 → 左心室（循环）

五、血型与输血

ABO血型系统：A型、B型、AB型、O型。

  • 输血原则：以输同型血为原则。紧急情况下可以异型输血，但必须慢、少量。
  • 万能供血者：O型血（红细胞上无A、B凝集原，可以输给任何血型）。
  • 万能受血者：AB型血（血浆中无抗A、抗B凝集素，可以接受任何血型）。
  • 健康成年人每次献血200~300毫升不会影响健康。
