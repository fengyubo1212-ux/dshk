第6章 人体生命活动的调节

一、眼球的结构与视觉的形成

眼球的结构

结构 位置 功能
角膜 眼球最前方 无色透明，光线透过
瞳孔 虹膜中央的圆孔 光线进入眼球的通道，可调节大小（强光下缩小，弱光下放大）
晶状体 瞳孔后方 双凸透镜状，折射光线，曲度可调节
玻璃体 晶状体后方 透明胶状物质，支撑眼球壁，折射光线
视网膜 眼球壁最内层 含感光细胞（视锥细胞和视杆细胞），能感受光的刺激并产生神经冲动

视觉形成过程

外界物体反射的光线 → 角膜 → 瞳孔 → 晶状体（折射） → 玻璃体（折射） → 视网膜（形成倒立缩小的物像，感光细胞产生神经冲动） → 视神经（传导神经冲动） → 大脑皮层的视觉中枢（形成视觉）。

注意：视网膜是成像的部位，大脑皮层视觉中枢是形成视觉的部位。视网膜受损→不能成像→失明；视神经或大脑皮层视觉中枢受损→能成像但无视觉→失明。

二、近视与远视

比较项目 近视 远视
成因 晶状体曲度过大，或眼球前后径过长 晶状体曲度过小，或眼球前后径过短
物像位置 物像落在视网膜前方 物像落在视网膜后方
矫正方法 配戴凹透镜 配戴凸透镜
预防 保持正确读写姿势，控制用眼时间，多户外活动 配戴合适的凸透镜矫正

三、耳的结构与听觉的形成

耳的结构

部位 结构 功能
外耳 耳郭、外耳道 收集声波，传导声波
中耳 鼓膜、听小骨（锤骨、砧骨、镫骨）、咽鼓管 鼓膜将声波转化为振动；听小骨将振动传导到内耳；咽鼓管连通鼓室和咽，保持鼓膜两侧气压平衡
内耳 耳蜗、前庭、半规管 耳蜗含听觉感受器，将振动转化为神经冲动；前庭和半规管感受头部位置变化，与平衡有关

听觉形成过程

声波 → 耳郭（收集） → 外耳道 → 鼓膜（产生振动） → 听小骨（传导和放大振动） → 耳蜗（听觉感受器产生神经冲动） → 听觉神经（传导神经冲劫） → 大脑皮层的听觉中枢（形成听觉）。

四、神经系统的组成

  • 中枢神经系统：包括脑（大脑、小脑、脑干）和脊髓。
  • 周围神经系统：包括脑神经（12对，由脑发出）和脊神经（31对，由脊髓发出）。
脑的组成 主要功能
大脑 有左右两个半球，表面为大脑皮层（灰质），有感觉、运动、语言、视觉、听觉等多种神经中枢，是调节人体生命活动的最高级中枢
小脑 使运动协调、准确，维持身体平衡
脑干 有专门调节心跳、呼吸、血压等人体基本生命活动的生命中枢

五、神经元的结构

神经元（又叫神经细胞）是神经系统结构和功能的基本单位。

  • 细胞体：含有细胞核，代谢和营养中心。
  • 树突：数量多而短，呈树枝状分支，接受刺激并将神经冲动传向细胞体。
  • 轴突：一般只有一条，较长，将神经冲动从细胞体传出。

功能：感受刺激 → 产生神经冲劫 → 传导神经冲动。

反射：人体通过神经系统，对外界或内部的各种刺激所发生的有规律的反应。反射是神经系统调节生命活动的基本方式。

六、反射弧

完成反射的结构基础是反射弧，一个完整的反射弧包括五个部分：

  1. 感受器：感受刺激，产生神经冲动。
  2. 传入神经：将神经冲动从感受器传至神经中枢。
  3. 神经中枢：接受传入的神经冲动，经过分析综合后产生新的神经冲动。
  4. 传出神经：将神经冲动从神经中枢传至效应器。
  5. 效应器：接受传出神经的冲动，作出反应（如肌肉收缩、腺体分泌）。

反射弧中任何一个环节受损，反射都无法完成。

反射的类型

比较项目 非条件反射 条件反射
形成 生来就有，先天性 在生活过程中逐渐形成，后天性
神经中枢 大脑皮层以下（脊髓、脑干等低级中枢） 必须经过大脑皮层（高级中枢参与）
稳定性 固定，不会消退 暂时、不固定，建立后可消退
意义 适应变化较小的环境 适应多变的环境
举例 缩手反射、膝跳反射、眨眼反射、吃梅子分泌唾液 望梅止渴、谈梅止渴、听到上课铃声进教室

七、激素调节

激素是由内分泌腺（无导管，分泌物直接进入血液）分泌的化学物质，在血液中含量极少，但对生命活动有重要的调节作用。

激素 分泌腺体 主要功能 分泌异常的疾病
生长激素 垂体 促进生长，调节蛋白质合成和骨骼生长 幼年不足→侏儒症；幼年过多→巨人症；成年过多→肢端肥大症
甲状腺激素 甲状腺 促进新陈代谢和生长发育，提高神经系统的兴奋性 幼年不足→呆小症；成年不足→甲减；过多→甲亢（甲状腺功能亢进）
胰岛素 胰岛（散布在胰腺中） 调节糖代谢：降低血糖浓度（促进血糖合成糖原，抑制非糖物质转化为糖） 分泌不足→糖尿病（血糖升高，尿中有糖）

重要知识点：侏儒症（生长激素不足）与呆小症（甲状腺激素不足）都是身材矮小，但侏儒症智力正常，呆小症智力低下。胰岛素是唯一降低血糖的激素。
