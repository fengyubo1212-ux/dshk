第7章 绿色植物与生物圈

一、光合作用

1. 概念与公式

光合作用是绿色植物通过叶绿体，利用光能，把二氧化碳和水转化成储存能量的有机物（淀粉等），并释放氧气的过程。

二氧化碳 + 水  ——光能→  有机物（储存能量） + 氧气
                   叶绿体

光合作��的场所是叶绿体，它既是生产有机物的"车间"，也是将光能转变为化学能的"能量转换器"。

2. 光合作用的原料、条件、产物和场所

要素 内容 说明
原料 二氧化碳、水 二氧化碳通过气孔进入，水由根从土壤中吸收
条件 光 光能提供动力（有光才能进行）
场所 叶绿体 含有叶绿素，能捕获光能
产物 有机物（淀粉）、氧气 有机物储存能量，氧气释放到大气中

3. 光合作用的意义

  • 制造有机物：为植物自身和其他生物（包括人类）提供食物和能量来源。
  • 转化并储存太阳能：将光能转变为化学能，储存在有机物中。
  • 维持碳-氧平衡：吸收二氧化碳，释放氧气，维持大气中二氧化碳和氧气的相对平衡。

4. 光合作用在农业生产上的应用

  • 合理密植：充分利用阳光，提高光合效率。
  • 间作套种：不同作物搭配种植，充分利用光能。
  • 适当增加二氧化碳浓度、延长光照时间、增加光照强度等均可提高光合效率。

二、呼吸作用

1. 概念与公式

呼吸作用是细胞利用氧气，将有机物分解成二氧化碳和水，并释放能量的过程。呼吸作用的场所是线粒体（细胞的"动力工厂"）。

有机物（储存能量） + 氧气  →  二氧化碳 + 水 + 能量
（场所：线粒体）

2. 呼吸作用的意义

  • 为生物体的生命活动提供能量（如细胞分裂、生长发育、物质运输等所需的能量都来自呼吸作用释放的能量）。

注意：呼吸作用与"呼吸"（肺的通气）不同——呼吸作用是所有活细胞中时刻进行的有机物分解并释放能量的过程。

三、光合作用与呼吸作用的比较

比较项目 光合作用 呼吸作用
场所 叶绿体（含叶绿素的细胞） 线粒体（所有活细胞）
条件 必须有光 有光无光都能进行
原料 二氧化碳、水 有机物、氧气
产物 有机物、氧气 二氧化碳、水
能量变化 储存能量 释放能量
联系 光合作用的产物是呼吸作用的原料，呼吸作用的产物是光合作用的原料；两者是相反的过程，但不是简单的逆转（场所、条件都不同）。

核心结论：光合作用是合成有机物、储存能量的过程；呼吸作用是分解有机物、释放能量的过程。两者相互对立又相互依存。

四、蒸腾作用

1. 概念

蒸腾作用是水分以气体状态从活的植物体表面（主要是叶片）散失到大气中的过程。主要通过叶片上的气孔进行。

2. 蒸腾作用的意义

  • 拉动水分和无机盐在植物体内的运输：蒸腾作用产生的拉力是植物吸收和运输水分、无机盐的主要动力。
  • 降低叶片温度：水在变为水蒸气时吸收热量，防止叶片被灼伤。
  • 促进生物圈的水循环：参与自然界的水循环。

五、绿色植物对生物圈的作用

  • 维持碳-氧平衡：通过光合作用吸收二氧化碳、释放氧气，维持大气中二氧化碳和氧气含量的相对稳定。
  • 促进水循环：通过蒸腾作用将水分以水蒸气的形式释放到大气中，增加大气湿度，促进降水。
  • 涵养水源、保持水土：植物的根能固定土壤，森林能涵养水源，防止水土流失。
  • 防风固沙：植物能减弱风力，固定流沙，防止土地荒漠化。

六、我国植被类型及面临的问题

我国主要的植被类型

  • 草原、荒漠、热带雨林、常绿阔叶林、落叶阔叶林、针叶林等。

面临的主要问题

  • 森林覆盖率低：人均森林面积不足世界平均水平的1/4。
  • 草原退化、沙化：过度放牧、不合理开垦。
  • 土地荒漠化：过度砍伐、气候变化等原因导致。

保护措施

  • 植树造林（3月12日为植树节）。
  • 退耕还林、退耕还草。
  • 建立自然保护区。
  • 减少污染，保护生态环境。

绿化祖国的意义：爱护植被、绿化祖国，是每个公民应尽的义务。
