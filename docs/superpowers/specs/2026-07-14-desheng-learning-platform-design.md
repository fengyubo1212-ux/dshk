# 地生会考学习平台 - 设计文档

## 概述

为2027年6月咸阳市地生会考（人教版）搭建的纯静态HTML学习平台。目标用户是一名2026年9月升初二的男生，从零基础开始备考，周期约10个月。

### 核心需求

- 手机浏览器可直接打开使用（file://协议兼容）
- 支持打印，隐藏导航元素
- 纯静态HTML，无需服务器
- 教材：人教版地理 + 生物

---

## 功能

### 知识点学习
每章一个HTML页面，图文并茂展示教材知识点。知识点按难度分3层：基础（七七上）、进阶（七下）、综合（八上/八下）。

### 课后练习
每章配套5-10道选择题，学完即时作答。提交后立即判对错，显示正确答案，错题自动加入错题本。

### 模拟考试
按学科生成整套模拟题（约50题），带倒计时，提交后出分。题目来源：历年真题改编 + 教材知识点覆盖。

### 错题本
基于localStorage持久化。支持按学科/章节筛选，重做错题，答对自动移除。显示错题日期、原始错误答案。

---

## 页面类型

| 页面 | 职责 |
|------|------|
| `index.html` | 首页：学科选择 → 年级 → 章节列表 |
| `*-chapter.html` | 知识点页：章节正文 + 开始练习入口 |
| `*-quiz.html` | 练习页：选择题，即时判分 |
| `exam.html` | 模拟考试页：选题、计时、出分 |
| `errors.html` | 错题本页：筛选、重做、管理 |

## UI设计

### 配色
- 主背景：白色 `#fff`
- 地理主题色：`#2b7a78`（蓝绿）
- 生物主题色：`#4a9c5d`（绿色）
- 正确色：`#27ae60`，错误色：`#e74c3c`
- 文字色：`#2c3e50`，辅助文字：`#7f8c8d`

### 字体
- 系统默认：`-apple-system, "Microsoft YaHei", "PingFang SC", sans-serif`
- 标题：加粗，1.5倍正文字号
- 正文：16px（移动端舒适阅读）

### 布局
- 顶部固定导航栏（3个tab：首页/错题/考试）
- 面包屑显示当前位置
- 内容区自由滚动
- 触控目标最小44×44px
- 最大内容宽度720px，居中

### 打印样式（media print）
- 隐藏：顶部导航栏、按钮、操作区
- 显示：知识点正文、题目内容
- 字体缩小至12pt，黑白打印

## 约束

- 纯静态文件，`file://`协议打开，不依赖HTTP服务器
- 所有数据内嵌HTML，不通过fetch加载外部JSON
- 错题本通过localStorage持久化，同源限制下各页面共享
- 移动端优先，最小支持320px宽屏幕
- 无外部依赖（CDN/CSS框架/JS库），一切本地自包含

---

## 技术选型

| 项目 | 选择 |
|------|------|
| CSS框架 | 无，手写CSS（移动端优先 + 打印样式） |
| JS框架 | 无，原生Vanilla JS |
| 数据持久化 | localStorage |
| 图标 | 用CSS绘制或Unicode符号，无图标库 |
